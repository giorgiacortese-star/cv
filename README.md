# GitHub Pages Setup

This repository is structured as a static GitHub Pages site.

## Publish

1. Push this repository to GitHub.
2. Open `Settings > Pages`.
3. Under `Build and deployment`, select `Deploy from a branch`.
4. Choose the `main` branch and the `/(root)` folder.
5. Save and wait for GitHub Pages to publish the site.

## Structure

- `index.html`: published entry page
- `assets/css/main.css`: site styles
- `src/`: non-published source material
- `archive/`: previous exported files kept for reference

If you want, I can also add a custom domain setup or a GitHub Actions workflow for automatic deployment.
